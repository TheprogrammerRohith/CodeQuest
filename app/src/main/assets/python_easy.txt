What is the output of the following code?
print("Hello" * 3)
a) Hello Hello Hello
b) HelloHelloHello
c) Hello 3
d) Error
b
The string "Hello" is repeated 3 times, resulting in "HelloHelloHello".

Which of the following is not a valid variable name in Python?
a) var_name
b) _varname
c) 2varname
d) varName
c
Variable names in Python cannot start with a digit.

What is the output of the following code?
x = [1, 2, 3, 4]
print(x[1:3])
a) [1, 2]
b) [2, 3]
c) [3, 4]
d) [1, 2, 3]
b
The slice x[1:3] returns elements from index 1 to 2 (not including 3).

Which of the following is used to create an empty set in Python?
a) {}
b) set()
c) []
d) ()
b
The function set() is used to create an empty set. {} creates an empty dictionary.

What is the output of the following code?
x = "abcdef"
print(x[::-1])
a) abcdef
b) fedcba
c) error
d) None
b
The slice x[::-1] reverses the string.

Which of the following statements is true for Python dictionaries?
a) They are immutable.
b) They are ordered.
c) They can contain duplicate keys.
d) They are accessed by keys.
d
Dictionaries are accessed by keys and cannot contain duplicate keys.

What does the following code print?
x = {1, 2, 3, 4}
x.add(4)
print(len(x))
a) 3
b) 4
c) 5
d) Error
b
Sets do not allow duplicate elements, so the length remains 4.

What is the output of the following code?
def add(x, y=10):
    return x + y
print(add(5))
a) 5
b) 10
c) 15
d) Error
c
The default value of y is 10, so add(5) returns 5 + 10 = 15.

Which of the following methods adds an element to the end of a list?
a) add()
b) append()
c) insert()
d) extend()
b
The append() method adds an element to the end of a list.

What will be the output of the following code?
x = [1, 2, 3, 4]
x.remove(3)
print(x)
a) [1, 2, 4]
b) [1, 2, 3]
c) [1, 2, 3, 4]
d) Error
a
The remove() method removes the first occurrence of the specified value (3 in this case).
