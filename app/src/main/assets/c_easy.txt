What is the output of the following C++ code?
int x = 5;
cout << x++ << " " << ++x;
a) 5 7
b) 6 7
c) 5 6
d) 6 5
a
x++ prints the current value of x (5) then increments it to 6. ++x increments it to 7 and then prints it.

Which of the following is a valid C variable name?
a) int
b) case
c) my_var
d) 123var
c
Variable names cannot start with numbers, and 'int' and 'case' are reserved keywords.

What does the following code output?
int x = 10;
x += 5;
cout << x;
a) 10
b) 5
c) 15
d) 20
c
The += operator adds 5 to the original value of x, resulting in x = 15.
